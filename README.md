# Llefià Music Fest

## Descripció

En aquesta pàgina web, tenim la festa anual **Llefià Music Fest** on podem trobar el horari, el formulari de contactes i el cartell d'aquesta festa.

### Estructura de fitxers

Els fitxers creats amb la seva estructura: 

-  index.html

-  README.md

-  style.css

-  /imagenes

    - concert2.jpg
-  /paginas

    - cartel.html
    - contacto.html
    - horario.html

### Instruccions d'ús

1. Al obrir l'arxiu, podem veure una breu informació sobre la quantitat d'escenaris que n'hi han per el barri de Llefià.
2. Després podem anar-nos a cartell, per veure sobre l'espectacle quins artistes arribaràn, quin escenari serà i enllaços per persones que volen escoltar música.
3. En horari, podrme veure el horari del festival depenent de l'hora n'hi haurà un artistà.
4. Finalment, tenim una formulari de contacte per si voleu donar idees, escoltar música, etc.
